# ShortLinks — консольное приложение создание коротких ссылок

**ShortLinks** — это консольное Java-приложение для управления короткими ссылками.

Программа позволяет:

- Создавать короткие ссылки
- Редактировать ссылки
- Устанавливать количество переходов
- Владеть несколькими ссылками

---
## Запуск проекта

### Требования

- Java **17+**
- Maven **3.8+**

### Сборка и запуск

```sh
mvn clean package
java -jar target/FinanceCLI.jar
```

Или через Maven:

```bash
mvn exec:java -Dexec.mainClass="art.FinanceControl"
```

### Запуск тестов

```sh
mvn test
```

## Принцип работы

| Слой | Что делает |
|------|------------|
| `model` | DTO + основные структуры данных |
| `service` | Чистая бизнес-логика |
| `port` | Абстракции для взаимодействия |
| `infra` | Реальные реализации (JSON-хранилище и т.д.) |
| `cli` | UI над логикой (меню, примеры, help) |
